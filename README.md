# 🛒 SQL Анализ E-Commerce продаж 2024

## Описание
Продуктовый анализ данных интернет-магазина за 2024 год с помощью SQL.
Датасет содержит 3,900 покупателей с информацией о покупках, подписках и поведении.

## Цель
Найти продуктовые инсайты через SQL запросы — сегментация покупателей,
анализ категорий, способов оплаты и сезонности.

## Инструменты
- Python, SQLite, pandas
- matplotlib
- SQL: GROUP BY, JOIN, CTE, Window Functions, CASE WHEN

## SQL запросы
- Базовые метрики — средний чек, количество покупателей, диапазон цен
- Выручка по категориям и топ-10 товаров
- Сегментация по полу, подписке, способу оплаты
- CTE — сегментация покупателей на лояльных, активных и новых
- Window Functions — RANK() топ-3 товара внутри каждой категории
- Анализ по возрастным группам

## Ключевые выводы
- **Clothing** — главная категория, выручка 104,264$ (вдвое больше Accessories)
- **Сезонность минимальная** — все сезоны дают 55-60k$, бизнес стабилен
- **51.6% покупателей лояльные** (25+ покупок) — высокий retention
- **Возраст не влияет на средний чек** — все группы тратят ~59-60$
- **Подписка не увеличивает средний чек** — разница всего 0.38$
